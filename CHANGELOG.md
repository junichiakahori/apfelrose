# 🌹 アプフェルローゼ 〜追憶のメロディ〜 - 変更履歴 (CHANGELOG)

### v1.1.2 (2026/06/21)
- **開発用ツール追加**
  - ダブルクリックするだけでローカル開発サーバー（Python3 http.server）を起動し、ブラウザで `http://localhost:3000` を自動的に開くためのランチャースクリプト `start_server.command` (Mac用) を追加。これにより、ローカルでも手軽にBGMファイル（mp3）の再生確認ができるように改善。

---

### v1.1.1 (2026/06/21)
- **バグ修正 (Audio)**
  - ローカルファイル (`file://` プロトコル) から直接ゲームを起動した際、自動再生ポリシーの制限により `AudioContext` の初期化がブロックされ、BGMが一切鳴らなくなるバグを修正。
    - 起動時の即時オーディオ初期化を廃止。
    - ユーザーが初めて画面をクリック/タップしたタイミングで `AudioContext` をアクティブ化するようにライフサイクルを改善。これにより、ローカル起動時やブラウザの強固な自動再生制限下でも確実に音が再生されるように修正。

---

### v1.1.0 (2026/06/21)
- **機能追加 & 音声システム拡張**
  - Suno AI等で作成した外部BGMファイル（`.mp3`）の読み込み・再生に対応。
    - プロジェクト内に `audio` フォルダを作成し、BGMファイルを半角英数字で配置・整理する設計に移行。
    - 音声合成システム（`SoundSynth`）をハイブリッド化。ファイルが存在するBGM（今回は通常ステージ用 `bgm_normal.mp3`）はファイルをデコードしてWeb Audio APIでシームレスにループ再生。
    - ファイルが存在しないBGM（メニュー、ボス、エンディング等）は、従来の自動合成シーケンサーで再生する堅牢な自動フォールバック機能を実装。

---

### v1.0.9 (2026/06/21)
- **調整**
  - ゲーム全体のマスター音量を `0.04` から `0.12` に引き上げ（約3倍）。より音がはっきりと聴こえるように音量バランスを最適化。

---

### v1.0.8 (2026/06/21)
- **機能追加**
  - ゲームプレイ中の一時停止（ポーズ）機能を追加。
    - HUD上部中央に「⏸」ボタンを追加し、クリックでゲームをポーズ可能に。
    - キーボードの `Esc` または `P` キーでもポーズの切り替えができるように対応。
    - ポーズ画面に「BGM ON」と「SE ON」のトグルスイッチを追加し、ゲーム中も個別に音量制御できるように修正。

---

### v1.0.7 (2026/06/21)
- **機能改善**
  - BGMとSE（効果音）の個別オンオフ機能を追加。
    - 音声トグルボタンを「BGM ON」と「SE ON」の2つの個別のチェックボックスに分割。
    - 音声合成システム（`SoundSynth`）内部で、BGM再生用とSE再生用のゲインノード（`bgmGain`, `seGain`）を分離し、独立して制御できるよう改修。

---

### v1.0.6 (2026/06/18)
- **UI改善**
  - スマホChromeアプリで上下が切れる問題を修正。
    - CSS `height: 100dvh`（Dynamic Viewport Height）を採用。アドレスバー・ナビゲーションバーを除いた実際の表示領域を基準にするよう変更。
    - `visualViewport` APIによる動的スケーリングを実装。スマホブラウザのUI表示切替（スクロールでアドレスバーが消える等）にも追従するよう改善。
    - ゲームエリアがアスペクト比 `9:16` を維持しながら画面内に収まるよう `CSS transform: scale()` で縮小表示するよう修正。

---

### v1.0.5 (2026/06/18)
- **機能追加**
  - ランキングバックエンドを `server.js`（Node.js/Express + ローカルJSON）から **Supabase**（クラウドPostgreSQL）に移行。
    - GitHub Pagesからサーバーなしで直接SupabaseのREST APIにアクセスするよう変更。
    - 同名プレイヤーは高スコアで上書き（UPSERT）されるようDB側で管理。
    - Supabase接続失敗時は従来通りLocalStorageへフォールバック。
  - LocalStorageのキーを `maid_stg_ranking` / `maid_stg_player_name` から `apfelrose_ranking` / `apfelrose_player_name` に変更（他ゲームとの混在防止）。

### v1.0.0 (2026/06/18)
- **新規リリース - 縦スクロール・ローグライトファンタジーシューティング「アプフェルローゼ」**
  - **ストーリーテリング**: 記憶消失のメイド「ロザリー・スカーレット」が失われた主人の記憶を取り戻すノベルパート付きのシューティング。
  - **Web Audio API 自動合成BGM**: 外部音声ファイルを使わずにブラウザ上でエモーショナルなコード進行（王道進行・カノン進行など）をリアルタイムに自動シーケンス演奏するシステムを搭載。
  - **ローグライトスキルシステム**: 記憶のリンゴ（経験値）を回収してレベルアップした際、3枚のランダムなスキルカード（ローズショット、ソーンレーザー、ボム、ホーミング、シールドなど）から自機を強化。
  - **かすり（Graze）システム**: 敵の弾にギリギリまで近づくことでボーナススコアおよび追加経験値を獲得できるゲームプレイ設計。
  - **グローバルランキング**: `server.js`（Node.js/Express）と通信してハイスコアを保存。通信エラー時はLocalStorageへ自動フォールバックする仕組みを搭載。
