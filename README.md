# アプフェルローゼ 〜追憶のメロディ〜 🌹🍎 (APFELROSE ~MELODY OF REMINISCENCE~)

記憶消失の少女「ロザリー・スカーレット」が、失われた記憶とメロディを取り戻すために異世界を彷徨う、縦スクロール・ローグライトファンタジーシューティングゲームです。

切ないストーリー展開と、Web Audio APIによるノスタルジックなコード進行の自動合成BGMがプレイヤーの心を揺さぶります。

---

## 🎮 ゲームの特徴

1. **記憶の回収＆ローグライト（ランダム強化）システム**：
   - 敵モンスター（記憶のバグや歪み）を倒してドロップする大好物の「記憶のリンゴ（XP）」を回収します。（ロザリーは現実世界ではリンゴアレルギーですが、この精神世界なら大好きなリンゴをいくらでも食べられます！）
   - レベルアップ時にゲームが一時停止し、ランダムに出現する3枚のスキルカードから1つを選択して自機を強化。
   - ローズショット、ソーンレーザー、ローズティーボム、ホーミングローズ、紅茶のシールドなど、優雅な薔薇・紅茶モチーフのスキルを組み合わせて戦います。

2. **「かすり（Graze）」システム**：
   - 敵の弾にギリギリまで近づく（かすらせる）と、キラキラした火花と共にボーナススコアと追加経験値を獲得！

3. **ローズボム**：
   - 画面全体の敵弾を一瞬で美しい花びらに変えて消し去り、すべての敵に大ダメージを与えるピンチ回避用ボム。

4. **エモいストーリーとダイナミックBGM**：
   - ゲーム開始時、ウェーブの合間、ボス出現前、ボス撃破後にノベル形式のストーリーテキストが再生され、洋館の秘密とロザリーの運命が少しずつ解き明かされます。
   - **完全内蔵シンセサイザー**：外部音源ファイルを一切使用せず、Web Audio APIだけでエモーショナルな進行（王道進行・カノン進行）のBGMをリアルタイムに自動シーケンス演奏します。ゲームのフェーズ（道中、ボス、レベルアップ、クリア）に合わせて曲調やテンポが滑らかに変化します。

5. **マルチプラットフォーム＆グローバルランキング**：
   - PC（キーボード/マウス）およびスマートフォン（タッチスワイプ、専用ボムボタン）のどちらにも完全対応。
   - ハイスコアランキングは、サーバーAPI（`server.js`）およびLocalStorageの両方に完全自動フォールバック対応しています。

## 🕹️ 操作方法

* **自機の移動**：
  - **【PC】** マウス移動（ドラッグ） または `W` `A` `S` `D` / 矢印キー
  - **【スマホ】** 画面をドラッグ
* **ボム発動**：
  - **【PC】** スペースキー または 画面クリック
  - **【スマホ】** 画面右下の「BOMB」ボタンをタップ
* **ストーリーの進行**：
  - タップ / スペースキーで次のセリフへ

## 🚀 遊び方（起動方法）

このゲームは、サーバー経由での起動、またはブラウザで直接ファイルをダブルクリックするだけでもプレイできます。

### 方法1：ローカルサーバーで動かす場合（推奨・ランキング保存対応）
Pythonがインストールされている環境であれば、以下のコマンドで簡易HTTPサーバーを立ち上げることができます。

```bash
# プロジェクトのルートディレクトリで実行
python3 -m http.server 3000
```
起動後、ブラウザで `http://localhost:3000` を開きます。

### 方法2：ダブルクリックして起動（超簡易プレイ）
`index.html` ファイルをブラウザ（Chrome, Safari, Edgeなど）にドラッグ＆ドロップ、またはダブルクリックして開くことで、サーバーなしですぐにプレイ可能です。（この場合、ランキングスコアはブラウザのLocalStorageに保存されます）

## 📂 ファイル構成
* `index.html` - UI、HUD、ストーリーダイアログ、モバイル用コントロール
* `style.css` - パステルネオンを基調としたネオンダークテーマスタイル
* `game.js` - メインゲームループ、当たり判定、かすり、ローグライト強化、BGM自動生成、効果音シンセサイザー
* `server.js` - スコア保存用の軽量Node.jsサーバー
* `.gitignore` - 不要ファイルの除外設定
